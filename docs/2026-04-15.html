For engineers working in security, this is the moment where the threat model fundamentally changes, and both Anthropic and OpenAI are trying to be your vendor for the defensive side.

WordPress Plugin Supply Chain Attack: Backdoors Found After Acquisition
Dozens of WordPress plugins were reportedly compromised after being acquired by a new corporate owner who then pushed backdoored updates to the existing install base, affecting thousands of websites that had these plugins installed and set to auto-update. This is a textbook software supply chain attack using a particularly clever vector: buy legitimate, trusted plugins with existing user bases, then use the update mechanism that users already trust to deliver malicious code. The attack is especially insidious because it exploits the trust relationship that's supposed to be a feature of the plugin ecosystem — the idea that updates from an existing plugin author are safe. For site operators, this is a reminder that any plugin with auto-updates enabled is implicitly trusting whoever currently owns that plugin, not just whoever wrote it originally. The WordPress ecosystem is particularly vulnerable to this pattern because plugin ownership changes hands relatively frequently and there's no robust mechanism for notifying users when a plugin is acquired. Security researchers are likely still untangling the full scope of what the backdoors do, but common payloads in these attacks include remote code execution capabilities, credential harvesting, and the ability to inject malicious content. If you maintain WordPress sites professionally, this is a good moment to audit which plugins have auto-updates enabled, check for any unusual plugin publisher changes in recent months, and review server logs for anomalous outbound connections.

NASA Is Building the First Nuclear-Powered Interplanetary Spacecraft
NASA administrator Jared Isaacman announced plans for the first nuclear reactor-powered spacecraft designed for interplanetary missions, timed to coincide with the Artemis II lunar flyby. This is a significant departure from the radioisotope thermoelectric generators (RTGs) that have powered probes like Voyager and Curiosity — those use passive radioactive decay to generate heat and electricity, while this would be an actual fission reactor generating substantially more power. More power unlocks mission capabilities that have been off the table: electric propulsion systems that are far more efficient than chemical rockets, the ability to send more instruments further and faster, and potentially crewed missions to Mars or beyond on timelines that don't require multi-year transits. The technical challenges are real — designing a reactor that operates reliably in space, passes launch safety reviews, and can handle the thermal environment of spaceflight is hard. The US has launched nuclear reactors in space before (the SNAP-10A in 1965) but hasn't done so since, so there's an institutional knowledge gap that needs to be rebuilt. Isaacman's announcement style — making it alongside Artemis II — suggests this is meant to signal a broader vision for an ambitious NASA agenda rather than just a single program. MIT Technology Review's explainer on the technology is worth reading for the engineering details. For space industry followers, nuclear propulsion has been 'coming soon' for decades; the question is whether this administration has the sustained political will to see it through to flight.
